# Loan-Risk-and-Performance-Analysis


## Overview

This project analyzes a financial loan dataset containing over 38,000 loan applications.

The goal is to identify funding trends, repayment patterns, borrower behavior, and lending performance using data analysis and visualization techniques.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Jupyter Notebook

---

## Key Performance Indicators

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income Ratio

---

## Analysis Performed

### State-wise Funding Analysis
Analyzed total loan amount funded across different states.

### Loan Term Analysis
Compared loan distributions across different loan terms.

### Purpose-wise Loan Analysis
Identified loan purposes contributing most to funded amounts.

### Home Ownership Analysis
Examined funding trends among Rent, Mortgage, and Own categories.

### Employment Length Analysis
Evaluated funding patterns based on borrower employment history.

### Repayment Analysis
Analyzed total amount received and repayment performance.
